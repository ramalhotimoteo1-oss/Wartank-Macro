# ⚔️ Wartank Macro (Termux)

Automação (macro/bot) para o jogo Wartank, desenvolvida para rodar diretamente no **Termux (Android)**.

---

## 📱 Requisitos

Antes de começar, atualize o Termux:

```sh
pkg update && pkg upgrade -y
Instale as dependências necessárias:
Bash
pkg install git curl bash sed grep -y
📥 Instalação
Clone o repositório:
Bash
git clone https://github.com/ramalhotimoteo1-oss/Wartank-Macro.git
Acesse a pasta do projeto:
Bash
cd Wartank-Macro
Dê permissão de execução aos scripts:
Bash
chmod +x *.sh
⚙️ Configuração
Edite os arquivos de configuração conforme necessário:
Bash
nano config.conf
ou:
Bash
nano accounts.txt
Preencha com seus dados corretamente.
▶️ Execução
Inicie o bot com:
Bash
./start.sh
ou:
Bash
bash main.sh
🛠️ Possíveis Problemas
Permissão negada
Bash
chmod +x *.sh
Comando não encontrado
Bash
pkg install bash
Erro de execução
Bash
dos2unix *.sh
🔄 Atualização
Para atualizar o bot:
Bash
git pull
❌ Remoção
Para remover o projeto:
Bash
rm -rf Wartank-Macro
⚠️ Aviso
Este projeto é apenas para fins educacionais.
O uso de automação pode violar os termos do jogo.
Use por sua conta e risco.
